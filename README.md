## Built With

- **HTML5**
- **CSS3**
  - CSS Grid (primary layout system)
  - Flexbox (component-level alignment)
  - Custom reset
  - Box shadows & borders for depth
- **SVG icons**

No JavaScript or external libraries were used.

---

## Features

-  Fully grid-based layout (no frameworks)
-  Sidebar navigation with icons
-  Header with search bar, notifications & user profile
-  Main content cards arranged using responsive grid
-  Right sidebar with announcements & trending users
-  Subtle shadows and borders for modern UI depth
-  Responsive behavior using `auto-fit` & `minmax`

---

##  What I Learned

- Proper use of **CSS Grid for page-level layouts**
- Difference between **grid alignment vs spacing**
- When to use **padding vs margin** in grid layouts
- Combining **Grid + Flexbox** effectively
- Creating reusable UI components
- Using `box-shadow` correctly for elevation
- Structuring semantic HTML for dashboards
